# Обзор проекта

## О проекте
B2B каталог кондитерских изделий. Компания **Naseer** производит и продаёт товары оптом магазинам. Сайт — витрина для покупателей (магазинов), позволяет ознакомиться с ассортиментом и оставить заявку.

Без оплаты, без авторизации, без корзины.

## Стек

| Слой | Технология |
|---|---|
| Фронтенд | Next.js 15, App Router, TypeScript, Tailwind CSS |
| i18n | next-intl (uz / en / ru) |
| Данные | JSON-файлы (products, lines) |
| Email | Resend — отправка с info@naseer.uz |
| Telegram | grammY или node-telegram-bot-api |
| Деплой | Vercel |
| Домен | naseer.uz (Cloudflare) |

## Языки

| Язык | Код | Направление | Статус |
|---|---|---|---|
| Узбекский | `uz` | LTR | В разработке (основной) |
| Английский | `en` | LTR | В разработке |
| Русский | `ru` | LTR | В разработке |
| Арабский | `ar` | RTL | Требует уточнения у клиента |

Маршруты: `/uz/...`, `/en/...`, `/ru/...`. Арабский требует отдельной RTL-адаптации дизайна.
