# Telegram бот

Отдельный бот для покупателей (магазинов) — альтернативный канал просмотра каталога и подачи заявок.

## Функции

- Просмотр линеек каталога через inline-кнопки
- Карточка товара: фото, описание, вес, упаковка
- Оформление заявки прямо в боте
- Менеджер получает заявки из бота в том же чате, что и с сайта

## Сценарий использования

```
Пользователь: /start
Бот: Добро пожаловать в Naseer! Выберите линейку:
     [Lollis] [Choco Cone] [Toffical] [Choco Bisco] [Bamboonee]

Пользователь: → Lollis
Бот: Линейка Lollis — карамельные конфеты и леденцы.
     Выберите товар: [Кола+Фанта+Мохито 440г] [Фруктовый микс 440г] ...

Пользователь: → Кола+Фанта+Мохито 440г
Бот: [фото] Lollis — Кола, Фанта, Мохито
     Вес: 440 гр | Упаковка: пачка
     [Оставить заявку]

Пользователь: → Оставить заявку
Бот: Введите название вашего магазина:
...
```

## Библиотека

**grammY** (`grammy`) — современная, TypeScript-first библиотека для Telegram ботов.

```ts
import { Bot, InlineKeyboard } from 'grammy'
const bot = new Bot(process.env.TELEGRAM_BOT_TOKEN!)
```

## ENV переменные

```env
TELEGRAM_BOT_TOKEN=       # токен бота (от @BotFather)
TELEGRAM_CHAT_ID=         # chat_id менеджера (для уведомлений о заявках)
```

## Деплой

Бот запускается отдельно от Next.js сайта. Варианты:
- Vercel Serverless Function (webhook)
- Отдельный Node.js сервис (polling)
