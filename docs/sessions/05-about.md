# Сессия 05: Редизайн страницы о компании

## Цель
Переписать `/about` под editorial-стиль. Это самая «текстовая» страница, главное — типографика и ритм.

## Контекст
- Файл: `src/app/[locale]/about/page.tsx`. Сейчас — стандартный hero + 3 advantages-карточки с unicode-иконками `◈ ◉ ✦` + FAQ + CTA. Иконки-символы выглядят дёшево.
- Реальный текст «о компании» от клиента ещё не получен — генерируй временный, копирайтер заменит.
- Дизайн-система та же.

## Что делать

### 1. Hero

- Editorial: `eyebrow` («О компании»), большой display-заголовок (генерируй, например «Мы делаем сладости для Ташкента с 2018 года.»). 
- `body-lead` подзаголовок — одна мысль из 1-2 предложений.
- Без brand-плашки.
- Справа — большая фотография (использовать одну из production/* как placeholder, или фото фабрики, если появится).

### 2. Story-секция

- Большой блок текста в editorial-стиле: левая колонка пустая (или с `eyebrow` «История» вертикально), правая — текст в 2-3 абзаца, ширина `max-w-2xl`, шрифт display для первого абзаца (lead), потом sans для остального.
- Внутри текста — крупная цитата (`<blockquote>` с display-section, кавычки декоративные).

### 3. Цифры компании

- Использовать `HomeStats` компонент (или сделать вариант) с другими цифрами: «2018 — год основания», «60 SKU», «5 линеек», «Tashkent — наш город». Тот же editorial-divide.

### 4. Advantages (заменить старые 3 карточки)

- Вместо `◈ ◉ ✦` карточек — **три горизонтальных блока друг под другом** с разделителями `border-y border-ink/10`:
  - Слева: тонкий номер `01/02/03` + display заголовок («Качество», «Ассортимент», «География»).
  - Справа: `body-lead` описание.
  - Между блоками — большие отступы.
- Никаких символов-иконок. Только типографика.

### 5. Линк на производство

- Тизер-блок (как на главной): фото factory-line + кнопка «Посмотреть производство».

### 6. FAQ

- Использовать существующий `<FAQ>` компонент (он уже работает на главной). Тот же тон.

### 7. Финальный CTA

- Тот же паттерн что на главной/production.

### 8. i18n

Использовать существующие `about.*` где можно. Добавить:
- `about.heroEyebrow`, `about.heroTitle`, `about.heroBody`
- `about.storyEyebrow`, `about.storyParagraphs` (массив строк или один длинный текст)
- `about.quote` — короткая фраза для blockquote
- `about.advantage1Title/Body`, `about.advantage2Title/Body`, `about.advantage3Title/Body`
- `about.productionTeaserTitle`, `about.productionTeaserCta`

## Definition of done

- [ ] Страница рендерится во всех локалях.
- [ ] Hero без brand-плашки.
- [ ] Старые иконки `◈ ◉ ✦` заменены на типографические блоки.
- [ ] Цифры показаны в editorial-стиле (`HomeStats` или его вариант).
- [ ] Линк на `/production` работает.
- [ ] Пометить сессию 05 как `done` в `session_roadmap.md`.

## Что НЕ делать

- Не выдумывать историю компании детально (даты, имена основателей) — генерируй общими словами, копирайтер заменит.
- Не добавлять «команда» / «фотографии сотрудников» — нет данных.
