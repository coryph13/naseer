# Сессия 03: Редизайн страницы производства

## Цель
Переписать `/production` под editorial-сторителлинг: страница должна работать как «sticky-scroll история» — большие фото, крупные цифры этапов, серьёзный тон без «хвалебных эпитетов».

## Контекст
- Текущий файл: `src/app/[locale]/production/page.tsx`. Сейчас — 4 этапа в чередующейся сетке `image|text` с framer-motion FadeUp. Структура норм, исполнение наспех.
- Фото в `public/production/` — Unsplash-заглушки (ingredients/factory-line/quality/packaging). Пользователь заменит позже своими съёмками. **Не удалять текущие** — оставить как placeholder.
- Сертификаты сейчас — три белые карточки с круглой иконкой-чек. Переделать под editorial-список документов.
- Дизайн-система та же что у главной.

## Что делать

### 1. Hero страницы

- Editorial-hero как на главной: `eyebrow` («Производство»), большой display-заголовок (генерируй: «Свой цех. Своё качество.» / «Our factory. Our standard.» / «O'z sexi. O'z standartimiz.»), `body-lead` подзаголовок с одной мыслью «Не перепродаём. Производим».
- **Без brand-плашки** в hero. Чистый белый/cream фон.
- Справа от текста — большая фотография `factory-line.jpg` или коллаж из двух (как в HeroVisual главной, но с production-фото).

### 2. Stages (4 этапа)

- Каждый этап — **полноширинная sticky-секция**:
  - Слева sticky колонка с большой цифрой `01–04` (классы `display-mono text-[160px] md:text-[240px] text-ink/8`) и `eyebrow` названием этапа.
  - Справа большая фотография этапа (full-height картинка, около 80vh).
  - Под секцией — текст этапа крупно (`display-section text-3xl md:text-4xl`) и описание `body-lead`.
- Между этапами — большой воздух (`py-32` или `py-40`).
- Можно использовать `motion-driven scroll progress` для подсветки активного этапа в боковой `eyebrow`-навигации (опционально, не блокирует DoD).

### 3. Сертификаты

- Заменить три карточки с иконкой-чек на **editorial-список документов**:
  - `eyebrow` «Сертификаты»
  - `display-section` заголовок
  - `body-lead` короткое описание
  - Список из 3 строк, каждая — серая горизонтальная плашка с разделителем сверху (1px `border-ink/10`):
    - Номер `01` тонкий слева
    - Название сертификата
    - Справа — «Предоставляется по запросу» мелким текстом или ссылка-плейсхолдер «Скачать PDF» (если файла нет — серым неактивным с tooltip «Скоро»)
- Без круглых иконок-чеков. Чистая типографика.

### 4. Финальный CTA

- Тот же паттерн что на главной: cream-фон, центрированный display-заголовок, кнопка-pill brand-цвета. Текст «Хотите производственный аудит / увидеть всё своими глазами?» — генерируй.

### 5. i18n

Все тексты — в `messages/*.json`. Использовать существующие `production.*` ключи где можно. Добавить:
- `production.heroEyebrow`, `production.heroTitle`, `production.heroBody` — новые
- `production.stageEyebrow` (например `Этап` / `Stage` / `Bosqich`) — для каждого этапа
- `production.certListTitle`, `production.certComing` — для нового списка
- `production.ctaTitle`, `production.ctaBody`, `production.ctaBtn` — для финального CTA (частично есть)
