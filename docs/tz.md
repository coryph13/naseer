# ТЗ — Naseer: сайт-каталог + Telegram бот

## О проекте
B2B каталог кондитерских изделий. Компания Naseer производит и продаёт товары оптом магазинам.

## Дизайн
- Основной цвет: `#8f1538` (тёмно-бордовый)
- Вторичный цвет: `#FFFFFF` (белый)
- Акцент по необходимости: светлый нейтральный (бежевый / светло-серый)

## Стек
- **Фронтенд:** Next.js 15, App Router, TypeScript, Tailwind CSS
- **i18n:** next-intl — поддержка uz / en / ru (маршруты `/uz/...`, `/en/...`, `/ru/...`)
- **Данные:** JSON-файлы (products, lines)
- **Email:** Resend (отправка через info@naseer.uz — Cloudflare Email Routing)
- **Telegram бот:** node-telegram-bot-api или grammY
- **Деплой:** Vercel

## Языки
| Язык | Код | Направление | Статус |

## Уведомления о заявках

При каждой заявке с сайта срабатывают два канала одновременно:
1. **Email** — на почту менеджера через Resend (info@naseer.uz)
2. **Telegram** — уведомление боту менеджера через Telegram Bot API

### Как подключить Telegram уведомления
1. Создать бота через @BotFather → получить `BOT_TOKEN`
2. Менеджер пишет боту `/start` → получить его `CHAT_ID`
3. При заявке — POST запрос к Telegram API:

```js
fetch(`https://api.telegram.org/bot${BOT_TOKEN}/sendMessage`, {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ chat_id: CHAT_ID, text: 'Новая заявка:\n...' })
})
```

`BOT_TOKEN` и `CHAT_ID` хранятся в `.env` переменных.
