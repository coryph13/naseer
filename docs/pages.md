# Структура страниц сайта

## Маршруты

| Страница | Путь | Статус |
|---|---|---|
| Главная | `/[locale]/` | ✅ реализована |
| Каталог (линейки) | `/[locale]/catalog` | ✅ реализована |
| Каталог (товары линейки) | `/[locale]/catalog/[line]` | ✅ реализована |
| О компании | `/[locale]/about` | ✅ скелет, нужен контент |
| Контакты | `/[locale]/contacts` | ✅ реализована |

Локали: `uz` (default), `ru`, `en`. Все маршруты локализованы через next-intl.

---

## Главная (`/`)

**Реализовано:**
- Hero-секция: слоган, вращающиеся кольца, пульсирующие градиенты, мини-превью товаров
- Статистика: 5 линеек / 62 позиции / поставки по Узбекистану
- Сетка линеек: кликабельные карточки → `/catalog/[line]`
- About-секция (тёмная, на бренд-цвете)
- CTA на контакты

**Ожидает контента:**
- Instagram лента (Behold.so, @naseer_uz, 6–9 постов)
- FAQ-секция (мин. заказ, доставка, сертификаты)
- Тексты от клиента для About

---

## Каталог линеек (`/catalog`)

SSR. Сетка из 5 карточек линеек: фото, название, описание, счётчик SKU, ссылка на линейку.

---

## Каталог товаров (`/catalog/[line]`)

SSR + client `ProductsGrid`.

**Реализовано:**
- Hero: название линейки, описание, breadcrumb
- Линейки-переключатель: горизонтальный ряд пилюль для быстрого перехода между линейками
- Фильтр по весу: пилюли с layoutId-анимацией
- Фильтр по вкусу: цветные точки-пилюли
- Карточки товаров (256px image): name/weight overlay на фото, цветной градиент вкуса, кнопка "Заявка"
- По клику на карточку — модальное окно (bottom sheet) с деталями: фото, вес, упаковка, CTA

**generateStaticParams:** 3 локали × 5 линеек = 15 страниц

---

## О компании (`/about`)

Скелет. Нужен контент: история, производство, сертификаты, география.

---

## Контакты (`/contacts`)

Форма заявки: имя, телефон, компания, сообщение.
Отправка через `/api/contact` → Telegram + Resend параллельно (Promise.allSettled).
